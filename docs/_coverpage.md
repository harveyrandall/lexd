# Lexd <small>0.1</small>

> A concise DSL for AT Protocol Lexicons — compile `.lexd` to lexicon JSON.

- Records, XRPC, subscriptions, permission sets
- Stdlib imports, decompiler, LSP & VS Code extension
- Works with goat, `@atproto/lex`, and Atmosphere tooling
